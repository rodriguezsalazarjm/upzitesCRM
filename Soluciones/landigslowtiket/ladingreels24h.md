## 1. Contexto general

UPZITES está creando una línea de productos low ticket. Cada producto tendrá su propia landing page independiente, con identidad visual propia, posible dominio o subdominio separado, y una experiencia enfocada en vender un servicio específico.

Este documento corresponde al producto:

# Reel Express 24H / Contenido en 24 Horas

Este producto representa la parte más visual, rápida, creativa, comercial y dinámica de UPZITES. Debe sentirse como una submarca independiente, pero conectada al ecosistema principal de UPZITES.

La promesa central del producto es:

**Grabamos, editamos y entregamos contenido vertical listo para publicar en 24 horas.**

El servicio está pensado para marcas, emprendedores, negocios locales, profesionales independientes, inmobiliarias, restaurantes, tiendas, servicios, marcas personales y creadores que necesitan contenido para redes sociales sin esperar semanas.

La landing debe funcionar como una vitrina visual del trabajo realizado, mostrando videos, casos de éxito, piezas UGC, reels comerciales, contenido para anuncios y ejemplos reales.

# 8. Requisitos técnicos sugeridos

Si se implementa en Next.js o React:

- Crear una estructura de datos editable para portafolio.
- Permitir agregar videos fácilmente desde un array o archivo de configuración.
- Cada video debe tener categoría, título, descripción, thumbnail, videoUrl y CTA.
- Usar lazy loading para videos.
- No cargar todos los videos pesados al inicio.
- Usar thumbnails y reproducir video solo al abrir modal o al hacer hover/tap.
- Optimizar imágenes y miniaturas.
- Crear componentes reutilizables para cards, planes, FAQs y portfolio.
- Mantener el diseño altamente responsive.
- Cuidar Core Web Vitals.

Ejemplo conceptual de estructura de portafolio:

```js
const portfolioItems = [
  {
    title: "Reel para restaurante",
    category: "Reels comerciales",
    objective: "Mostrar producto y generar reservas",
    thumbnail: "/portfolio/restaurante-thumb.jpg",
    videoUrl: "/portfolio/restaurante.mp4",
    cta: "Quiero un video así"
  },
  {
    title: "UGC producto belleza",
    category: "UGC",
    objective: "Mostrar uso real y beneficio del producto",
    thumbnail: "/portfolio/ugc-belleza-thumb.jpg",
    videoUrl: "/portfolio/ugc-belleza.mp4",
    cta: "Quiero contenido UGC"
  }
]
```
